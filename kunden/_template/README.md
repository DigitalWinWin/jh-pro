# Kunden-Template

Diesen Ordner als Vorlage für neue Kunden verwenden.

## Verwendung
1. Ordner kopieren und umbenennen: `[bereich]_[kundenname]` (z.B. `pb_praxis-mueller`)
2. Diese README.md löschen
3. Protokolle, Konzepte und Dokumente in die jeweiligen Unterordner

## Struktur
```
[bereich]_[kundenname]/
  protokolle/     ← Meeting-Protokolle, chronologisch
  konzepte/       ← Digitalisierungskonzepte, Maßnahmenpläne
  dokumente/      ← Angebote, Berichte, sonstige Unterlagen
```

## Namenskonvention
- Protokolle: `YYYY-MM-DD-[thema].md`
- Konzepte: `[typ]-[kundenname]-v[version].md`
- Dokumente: `[typ]-[kundenname]-[datum].md`
