KI-Arbeitskonzept — Die 3 Säulen

Praxisschulung Juliane Hoffmann
17. März 2026 · Digital WinWin
📚
Säule 1

Kontext­engineering

  • CLAUDE.md = Projektanweisung (sagt der KI, wer du bist)
  • Basisdokumente = dein Fachwissen als Kontext
  • Ordnerstruktur = Projektwissen (Kunden, Skills, Vorlagen)
  • Kontextdaten aktuell halten — nicht zumüllen
  • Markdown-Format gibt Struktur (wie Überschriften in Word)
«Die Mappe, die du einem neuen Mitarbeiter mitgibst.»
⚙
Säule 2

Skill­erstellung

  • Skill = wiederholbarer Prozess (wie eine Checkliste)
  • Besteht aus: SKILL.md + References + Assets
  • Erstellen via Skill Creator in Claude
  • Iterativ: testen → Feedback → anpassen
  • Immer lokal herunterladen und im Ordner ablegen
  • Beispiele: Coaching-Protokoll, Instagram-Karussell
«Ein Skill beschreibt deinen Prozess — die Chefkoch-Datei.»
✎
Säule 3

Prompt­verbesserung

  • Meta-Prompt: «Verbessere folgenden Prompt, gib dir Mühe, stelle mir Rückfragen»
  • Klare Zielvorstellung > perfekter Prompt
  • Ziel definieren → Prompt verbessern lassen → Kontext mitgeben
  • Rückfragen einfordern für bessere Ergebnisse
  • Ergebnisse als neue Vorlage zurückführen
«Du musst nicht gut prompten — du brauchst eine klare Zielvorstellung.»
Ziel definieren → Prompt verbessern → Kontext geben → Ergebnis
Agentische KI arbeitet auf dein Ziel hin —
gib ihr Kontext, Prozesse und klare Ziele.
