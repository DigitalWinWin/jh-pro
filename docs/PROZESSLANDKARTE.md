## Cross-Selling-Pfade

Die Produktkette definiert 4 systematische Cross-Selling-Pfade:

### Pfad 1: VFD → Medikit (PF → PB)
```
VFD-Mandat abgeschlossen
  → Praxis hat Strukturbedarf erkannt
    → Angebot Medikit Starter oder Profi
      → Beratungsprojekt PB
```
**Trigger:** VFD-Abschlussgespräch, Praxis äußert Wunsch nach "mehr Struktur"
**Potenzial:** Hoch (5 aktive VFD-Mandate, aktuell 0% systematisch genutzt)

### Pfad 2: BAFA → Medikit (PF → PB)
```
BAFA-Bericht erstellt
  → Maßnahmenplan enthält Digitalisierungsempfehlungen
    → Angebot Medikit für Umsetzung
      → Beratungsprojekt PB
```
**Trigger:** Maßnahmenplan-Präsententation
**Potenzial:** Mittel (abhängig von BAFA-Volumen)

### Pfad 3: ZEN → Medikit (DF → PB)
```
ZEN-Konzept durchlaufen
  → Praxis will weiter optimieren
    → Juliane als persönliche Beraterin empfehlen
      → Erstgespräch PB
```
**Trigger:** ZEN-Abschluss oder Check-in Call
**Potenzial:** Mittel (Dr. Flex muss einverstanden sein)

### Pfad 4: Medikit → Retainer (PB → PB)
```
Medikit-Projekt abgeschlossen
  → Praxis braucht laufende Betreuung
    → Retainer-Vereinbarung (185 EUR/h)
      → Regelmäßige Calls + Updates
```
**Trigger:** Abschlussgespräch Medikit-Projekt
**Potenzial:** Sehr hoch (natürliche Fortsetzung)
