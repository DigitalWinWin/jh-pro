## 3. Kernleistungen

### 3.1 Verfahrensdokumentation (VFD)
- **Pflicht laut GoBD** (seit 2015) für Betriebsprüfungen
- **Tool:** EasyDoku (spezielles VFD-Erstellungsprogramm)
- **Ablauf (4 Termine à 90 Min.):**
  1. Geschäftsprozesse festlegen
  2. Belegarten pro Geschäftsprozess aufschlüsseln
  3. Formulare ausfüllen im EasyDoku
  4. Entwurf der VFD an Mandant zur Prüfung
- **Vorerfassung:** MS Forms für Stammdaten (Hardware, Software, IT-Dienstleister)

### 3.2 BAFA-Förderberatung
- Mandanten sind BAFA-förderfähig
- **Maßnahmenplan erstellen** (KI-gestützt mit Master Prompt):
  - Struktur: Einleitung → Maßnahmenkatalog (Ist → Risiko → Empfehlung) → Umsetzungsfahrplan → Fazit
  - Stilvorgaben: Fließtext, keine Listen/Tabellen, praxisnah, lösungsorientiert
  - BAFA-konform: Individuell, klare Umsetzungsanleitungen, Rollen/Ressourcen/Zeitrahmen, QM-Fokus bei Heilberufen
  - Neutralität: Keine Produkt-/Anbieternamen, nur Gattungen
  - Umfang: ~5.000 Zeichen
- **Umsetzungsfahrplan:** kurzfristig / mittelfristig / langfristig
- **Checkliste der To-Dos** (nach Maßnahmen-Themen)
- BAFA-Antrag: Fragebogen, Dokumente, E-Mail (Kollegin/Jurastudentin betreut)

### 3.3 Beratungsbericht
- Formaler Bericht für BAFA-Nachweis
- **Struktur:** Darstellung Unternehmen → Anlass → Beratungszeitraum → Durchführung → Maßnahmenplan → Fazit
- **Beratungsumfang:** 1 Beratungstag / 3-4 Analysetermine, je 90 Min.
- **Bereiche:** Wirtschaftlich, finanziell, organisatorisch, personell
