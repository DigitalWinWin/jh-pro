## 4. FEHLENDE FUNNELS — Analyse

### 4.1 Praxisberatung (PB) — Aktueller Funnel

```
[Webinar/Empfehlung] → [Calendly-Buchung] → [Erstgespräch] → [Angebot] → [Beratung]
                          ↑ kein Nurturing       ↑ kein System    ↑ manuell
```

**Fehlend:**
- Kein Lead Magnet (kein Freebie zum Download)
- Kein Newsletter / E-Mail-Serie
- Kein systematisches Follow-up nach Erstgespräch
- Kein Testimonial-System (5 auf Website, aber nicht aktiv gesammelt)

### 4.2 Praxis Factory (PF) — Aktueller Funnel

```
[Steuerkanzlei-Empfehlung] → [Honorarrechner (versteckt)] → [Angebot] → [VFD-Prozess]
                                         ↑ nicht proaktiv genutzt
```

**Fehlend:**
- Kein aktives Marketing (nur Steuerkanzlei-Zuführung)
- Instagram "stiefmütterlich"
- Kein Webinar zum Thema VFD-Pflicht
- Kein Cross-Selling-Pfad VFD → Praxisberatung

### 4.3 Dr. Flex (DF) — Aktueller Funnel

```
[Dr. Flex Kunden] → [Support-Bedarf] → [Support-Videos] → [Zen-Konzept-Pitch]
                                                              ↑ noch kein Kunde bis KI-Videos
```

**Fehlend:**
- Kein eigener Funnel (alles über Dr. Flex)
- Videokurs-Idee existiert, aber kein Business Case
- Instagram-Vermarktung gewünscht aber nicht umgesetzt
