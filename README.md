# KI-Arbeitsumgebung — Juliane Hoffmann

Zentrale Arbeitsumgebung für Claude (Cowork / Desktop).

## Schnellstart

1. Öffne diesen Ordner in Claude Desktop / Cowork
2. Claude liest automatisch `CLAUDE.md` und kennt alle Basisdokumente
3. Starte mit einem Prompt wie: "Erstelle mir ein Kundenbriefing für Praxis Müller"

## Ordnerstruktur

- `basisdokumente/` — 38 Wissensdokumente über alle 6 Tätigkeitsbereiche
- `skills/` — 13 Skills für wiederkehrende Aufgaben
- `kunden/` — Kundenordner mit Protokollen, Konzepten, Dokumenten
- `vorlagen/` — Templates für Angebote, Protokolle, Berichte
- `brand/` — Logos, Farben, Brand-Assets
- `mcp-konfiguration/` — MCP-Server Setup-Anleitungen
- `docs/` — Dokumentation und Anleitungen

## Dokumentation

- `docs/SETUP.md` — Einrichtungsanleitung
- `docs/DAILY-USAGE.md` — Tägliche Nutzung und Beispiel-Prompts
- `docs/PROZESSLANDKARTE.md` — Alle Prozesse über die 6 Tätigkeitsbereiche
- `docs/SKILL-UEBERSICHT.md` — Übersicht aller 13 Skills
